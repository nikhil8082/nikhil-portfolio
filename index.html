Alcohol Detection Vehicle Speed Regulator

Designed microcontroller-based safety system using MQ-3 sensor and ESP8266 for real-time breath alcohol detection. Integrated LCD display, motor driver, relay circuits, and GSM module for emergency alerts. Enforces tiered speed limits (60/40/15 km/h) based on alcohol levels.

Embedded C IoT Safety Systems
